## 🚀 Getting Started

1. **Clone a Repo**

   ```bash
   git clone https://github.com/Open-Source-Bots-org/bot-name.git
   cd bot-name
   ```

2. **Install Dependencies**

   ```bash
   npm install  # or pip install -r requirements.txt depending on the bot
   ```

3. **Run the Bot**

   ```bash
   npm start  # or python bot.py
   ```
